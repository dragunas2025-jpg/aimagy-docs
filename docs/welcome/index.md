# Welcome to Aimagy Docs

Aimagy builds AI image generation and editing tools for creative applications.
This documentation covers installation, API setup, everyday workflows, and
troubleshooting.

## Choose your plugin

- [Flash Banana for Krita](/krita/flash-banana/overview/) includes Free and Pro editions.
- [AI Generator for Photoshop](/photoshop/ai-generator/overview/) is a separate Photoshop plugin.

Future Krita tools, including an upscaler and a dedicated image generator, will
be documented here as they are released.
