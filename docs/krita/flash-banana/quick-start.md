# Quick start

1. Install and enable Flash Banana in Krita.
2. Add the API key required by your selected provider.
3. Open or create a document.
4. Enter a prompt and select the desired operation.
5. Generate the result and continue editing in Krita.

See [API keys](./api-keys.md) before your first generation.
