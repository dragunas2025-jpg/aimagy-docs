# Flash Banana for Krita

Flash Banana brings AI image generation and editing workflows into Krita.
It is available in Free and Pro editions.

- **Free** provides the core workflow with some features locked.
- **Pro** provides the complete Flash Banana feature set.

Continue with [Free vs Pro](./free-vs-pro.md) or [Installation](./installation.md).
